# 中文代码审查 Prompt

使用场景：让 Claude Code 对当前代码进行中文 Code Review。

---

## Prompt 内容

```
请对以下代码进行 Code Review，用中文输出，按如下格式：

## 严重问题（必须修复）
- [文件:行号] 问题描述 → 建议修复方案

## 重要问题（建议修复）
- [文件:行号] 问题描述 → 建议修复方案

## 优化建议（可选）
- [文件:行号] 问题描述 → 建议修复方案

## 亮点
- 值得保留或学习的写法

审查维度：
1. 正确性：逻辑错误、边界条件、并发安全
2. 安全性：SQL 注入、XSS、敏感信息泄露、权限校验
3. 性能：N+1 查询、大对象序列化、不必要的循环
4. 可维护性：命名清晰、职责单一、重复代码
5. DDD 规范（Java 项目）：分层是否正确、事务位置、领域逻辑是否泄漏到非领域层
```

---

## 使用方式

在 Claude Code 中：
1. 选中要审查的代码
2. 输入 `/review` 或 `@code-review-cn 审查这段代码`

或直接在对话中粘贴代码，然后使用上面的 Prompt。
